# 🤖 Telegram Bot для анонимных вопросов

Бот для Telegram, который позволяет пользователям получать анонимные вопросы по персональной реферальной ссылке и отвечать на них.

---

## 🚀 Возможности

- 🔗 Генерация персональной ссылки для анонимных вопросов  
- 📩 Отправка анонимных сообщений  
- 💬 Возможность анонимного ответа  
- 👮 Админ-панель  
- 🚫 Блокировка / разблокировка пользователей  
- 📂 Выгрузка базы сообщений и рефералов  
- 📝 Логирование всех отправленных сообщений  

---

## 🛠 Используемые технологии

- Python 3.10+  
- python-telegram-bot  
- asyncio  
- nest_asyncio  
- Хранение данных в `.bd` файлах

## 🔐 Безопасность

❗ Никогда не публикуйте токен бота в открытом репозитории.

Рекомендуется использовать переменные окружения:

```python
import os
from telegram.ext import ApplicationBuilder

TOKEN = os.getenv("BOT_TOKEN")
app = ApplicationBuilder().token(TOKEN).build()
```

Запуск:

```bash
export BOT_TOKEN=your_token_here
python bot.py
```

## 👮 Админ-функции

Администратор (по `ADMIN_ID`) может:

- Заблокировать пользователя  
- Разблокировать пользователя  
- Получить файл `send.bd`  
- Получить файл `ref.bd`  
- Получить читаемый список сообщений  

Команда для входа в админ-панель:

```
/adm
```

---

## 📌 Как работает бот

### 1. Пользователь запускает `/start`

Получает персональную ссылку вида:

```
https://t.me/YourBot?start=USER_IDQ
```

### 2. Другой пользователь переходит по ссылке

Может отправить анонимный вопрос.

### 3. Получатель

- Получает вопрос  
- Нажимает кнопку **"Ответить"**  
- Ответ отправляется анонимно

## 💾 Хранение данных

### ref.bd

Формат:

```
username: user_id: ref_code
```

### send.bd

Формат:

```
@sender to @getter: сообщение (Message ID: chat_id.message_id)
```
